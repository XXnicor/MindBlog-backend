## Quando Tiver os GIFs Prontos:

Simplesmente substitua os placeholders pelos caminhos reais:

```markdown
<img src="docs/gifs/login-register.gif" alt="Demonstração de Login e Cadastro" width="800px">
```

E faça commit:

```bash
git add docs/gifs/*.gif
git commit -m "Adiciona GIFs de demonstração das funcionalidades"
git push
```
